Aide-mémoire

  • Le bouton de configuration sur l'écran Accueil vous permet de configurer la section « Aller à ».
  • Vous pouvez développer et réduire les volets pour afficher/masquer des détails supplémentaires, lorsque cet icone apparaît.
  • Saviez-vous que le texte souligné dans les vues détaillées représente une action de lien (telle que naviguer jusqu'à une URL ou accédez à un compte) ?
  • L'option « Effacer le stockage » sous l'option Paramètres ne sert qu'à des fins de dépannage pour effacer la configuration stockée localement.

Aide-mémoire des actions

  • Suite à une action téléphonique ou d'e-mail, une vue s'affiche, qui vous permet de saisir des notes et de les enregistrer dans l'historique.
  • L'action « Afficher l'adresse » vous renvoie à l'application de cartographie native sur les périphériques pris en charge.
